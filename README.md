# EETEPA Vilhena Alves

Site estático premium em Português-BR para apresentar, como **demo não oficial**, uma visão tecnológica da EETEPA Vilhena Alves.

**Demo:** https://albertomateus9.github.io/eetepa-vilhena-alves/

> Projeto demonstrativo não oficial inspirado na EETEPA Vilhena Alves; não representa canal institucional oficial.

## Visão Geral

O site usa HTML, CSS e JavaScript puro, sem backend, login, banco de dados ou envio de formulários. A proposta visual combina campus tecnológico amazônico, cultura maker, ensino técnico e portfólio educacional.

## Páginas

- `index.html`: página inicial com hero, destaques e chamadas.
- `cursos.html`: áreas formativas demonstrativas com filtros.
- `laboratorios.html`: cultura maker, robótica, impressão 3D e programação.
- `projetos.html`: vitrine de projetos educacionais com filtros.
- `contato.html`: localização pública, fontes e formulário local demonstrativo.

## Funcionalidades

- Menu mobile.
- Alternância de tema/contraste com `localStorage`.
- Filtros de cursos e projetos.
- Cards expansíveis.
- Formulário local sem envio para servidor.
- Botão para copiar endereço.
- Destaque automático da página atual.

## Fontes Públicas

- Agência Pará: certificação no Laboratório Maker da EETEPA Vilhena Alves.
- Agência Pará: Caravana de Ciência e Tecnologia em Belém.
- Agência Pará: contexto sobre educação técnica e tecnológica no Pará.

## Política De Dados

O site não publica dados reais de estudantes, documentos internos, telefones pessoais, e-mails pessoais, documentos de identificação, credenciais ou materiais privados. O formulário é demonstrativo e roda somente no navegador.
